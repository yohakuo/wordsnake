# 🐍 贪吃蛇猜单词 (Snake Word Game)

![Next.js](https://img.shields.io/badge/Next.js-15-black)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4.0-38bdf8)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)

玩家通过控制贪吃蛇吃掉正确的字母来拼写目标单词。

## 🎮 游戏玩法

### 核心机制
- **猜单词**：每局游戏会给出一个英语单词的提示（Hint）和部分已揭示的字母。
- **吃字母**：控制贪吃蛇在网格中移动，寻找并吃掉能填补单词空缺的**正确字母**。
- **连击系统 (Combo)**：连续快速吃到正确字母可获得额外分数奖励！
- **智能复习**：系统会自动记录你拼错的单词，并在后续游戏中优先出现，助你攻克难点。
- **躲避危险**：
  - 避免撞到墙壁。
  - 避免撞到蛇身。
  - 避免吃掉**错误字母**（会扣除生命值）。
- **生命值**：初始拥有 2 点生命值。吃错字母会扣除 1 点。生命归零则游戏结束。
- **得分系统**：
  - 吃对字母得分，连续通关分数会**累积**，自动记录本地**最高分**。

### 胜利与失败
- **胜利**：成功填满单词的所有字母。
- **失败**：撞墙、撞身体或生命值耗尽。

### ⌨️ 操作指南
- **桌面端**：
  - **移动**：使用键盘方向键 `↑` `↓` `←` `→` 控制。
  - **暂停/继续**：按 `P` 键。
- **移动端**：
  - **触屏控制**：支持手指滑动（Swipe）控制方向。

### ⚙️ 自定义词库
点击右上角的 **设置** 按钮，你可以导入自己的单词库进行练习。
- 支持 **JSON 格式** 和 **文本格式** (单词+空格+提示)。

## 🛠️ 技术栈

本项目基于现代化的前端技术栈构建：

- **核心框架**: [Next.js 16](https://nextjs.org/) (App Router)
- **开发语言**: [TypeScript](https://www.typescriptlang.org/)
- **UI 框架**: [Tailwind CSS 4](https://tailwindcss.com/)
- **组件库**: [shadcn/ui](https://ui.shadcn.com/) (基于 Radix UI)
- **图标库**: [Lucide React](https://lucide.dev/)
- **交互动画**: Framer Motion (粒子特效、屏幕震动)
- **音频处理**: Web Audio API

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```
启动后访问 [http://localhost:3000](http://localhost:3000) 即可开始游戏。

### 3. 构建生产版本
```bash
npm run build
```
